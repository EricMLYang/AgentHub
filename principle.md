# 2026 用好 AI 的領先指標

## 基礎工：Vibe Code (肌肉記憶)

1. 額度清零：GitHub Copilot 每月額度（或 API 配額）必須用完，確保高頻率的磨合。

2.AI-First 習慣：任何任務（寫代碼、改 SQL、修 README）想到就丟給 AI 做第一版，絕不讓 AI 閒著。

3. 高品質 Context：練習在對話前主動提供 Schema 或文檔（例如 Data Pipeline 的架構圖），提升「一次到位」的機率。

## 進階：Agent 化 (流程自動化)
1. 捕捉重複：每週列出一個超過 15 分鐘的重複性瑣事（如整理日誌、分析數據異常）。

2. 工具化部署：將這些瑣事轉化為專屬 Agent。目標是：手動操作次數逐月下降。

3. 知識庫化 (RAG)：將你的架構筆記、技術標準餵給 AI，讓 Agent 具備你的「專業視角」來回覆。
